Web Technologies

HTML

HTML is a scripting language for creating the base of every web page. With HTML, you can implement stlye within the HTML file or directly reference a CSS file. Also, JavaScript can be included to add dynamic aspects for a webpage.

CSS

CSS or cascading style sheets is a language to design HTML content. CSS is the presentation to the user of the HTML content. CSS can either be directly referenced in an HTML file with a style tag or imported with an external file tag in the HTML file.

JavaScript

JavaScript is a programming language that is used with HTML to create dynamic aspects of a webpage. JavaScript can add animation, and interactive properties to an HTML page. JavaScript can also be used to change or revise HTML and CSS content.

Python

Python is a high level programming language. Python is mostly used for data manipulation, creating websites, and web or mobile applications. Python is popular due to the simple syntax, leading more beginner programmers to learn Python and make amazing applications.

SASS

SASS is a CSS pre-processor. It is used to write a stlye sheet for HTML content in SASS and then transformed into CSS before it is rendered by a browswer. SASS is powerful due to extended features like variables, nested rules, or mixins. These features allow compling CSS much easier and more effienctly.

GitHub

GitHub is an online code storing, sharing, or posting website. GitHub can be used with Git, a version control system. It is simple to use Git in a terminal command to add your code to a GitHub repository.
